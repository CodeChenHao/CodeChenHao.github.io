# Johnny的空间站

> 命运就像是自己的掌纹,虽然弯弯曲曲,却永远掌握在自己的手中

一个基于 Hexo + AnZhiYu 主题构建的个人技术博客，分享编程学习笔记、技术教程和个人感悟。

- **所在地**: 中国，成都市

## ✨ 项目特点

- 🎨 **现代化设计**: 使用 AnZhiYu 主题，支持深色/浅色模式自动切换
- 📱 **响应式布局**: 完美适配桌面端和移动端
- 🔍 **全文搜索**: 集成 Algolia 搜索引擎，快速查找文章
- 💬 **评论系统**: 基于 Twikoo 的评论功能，支持弹幕效果
- 🎵 **音乐播放**: 内置音乐播放器，支持背景音乐
- 🌓 **主题切换**: 支持繁简体中文转换
- ⚡ **无刷新加载**: 使用 Pjax 技术，提升页面切换体验
- 🖼️ **图片灯箱**: 集成 Fancybox，优雅展示图片
- 📝 **代码高亮**: 使用 Highlight.js，支持多种编程语言

## 🚀 快速开始

### 本地运行

1. 克隆仓库
```bash
git clone https://github.com/CodeChenHao/CodeChenHao.github.io.git
```

2. 安装依赖
```bash
npm install
```

3. 启动本地服务器
```bash
hexo server
```

4. 访问 http://localhost:4000

### 部署到 GitHub Pages

1. 修改配置文件中的部署设置
2. 运行部署命令
```bash
hexo deploy
```
