## 一、什么是Linux

```shell
Linux,全称GNU/Linux,由林纳斯·本纳第克特·托瓦兹 编写的操作系统内核，是一套免费使用和自由传播的类Unix操作系统。支持多用户、多任务、多线程和多CPU
```

## 二、Linux的特点

```
1.一切皆文件
2.开源免费
3.多用户多任务
4.支持多平台
5.兼容POSIX.0标准
6.稳定
```

## 三、Linux应用场景

```
相比windows系统,更加稳定，因此主要用于服务端
```
